# Pedro-e-Ines---Lenda
Site interativo sobre a lenda de Pedro e Inês (século XIV). Inclui livro 3D, confessionário com 10 personagens históricos, legado cultural (lugares, livros, arte, audiovisual) e quiz educativo. Design responsivo com estética medieval e navegação intuitiva. Destinado a estudantes, turistas e entusiastas de história portuguesa.
